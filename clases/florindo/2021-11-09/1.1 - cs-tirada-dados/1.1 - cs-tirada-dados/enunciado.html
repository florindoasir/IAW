Tirada de dados - Enunciado

En este ejercicio se debe crear un programa que imite el funcionamiento básico de una tirada de 5 dados.

  • El programa tiene una página php:
    • Al cargar o recargar la página, debe mostrar cinco dados de seis caras:
      • Se deben mostrar cinco dados al azar.
      • Un botón permite volver a tirar los cinco dados. Enlace o recarga a la página actual
      • En cada tirada se muetra la puntuación total (sumando los cinco valores).
  • No se deben cambiar el nombre del control incluido en las plantillas.
  • Al actualizar la primera página, los dados se volverán a lanzar y cambiar los valores.
